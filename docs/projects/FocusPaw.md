## 项目简介

FocusPaw 是渊渟集团的第一个测试项目，主要目标是**验证多 Agent 协作工作流的完整性**。

项目特点：
- **非商业项目**：不做实际产品，只做 workflow 验证
- **全链路测试**：从 PRD 撰写到上线复盘的全流程
- **通信机制验证**：验证双通道通信方案（sessions_send + Discord）
- **Session 隔离测试**：验证多 Agent 在多 Session 环境下的协作

---

## 技术栈

- **前端**: HTML + Tailwind CSS (CDN)
- **后端**: 无（静态网站）
- **基础设施**: 
  - OpenClaw Gateway
  - Discord Bot API
  - 多 Agent 协作系统
